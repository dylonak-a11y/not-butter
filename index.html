GUARDIAN CLOCK

Destiny 2 time-played tracker

Enter any player's Bungie Name (the name + 4-digit code shown in-game). Their Destiny stats must be public.
Add up to 6 players, then Compare. Click a column header to sort, or a player's name to open their Track page. “Detailed” adds season/streak/session columns and each player's full breakdown.
First-time setup & how to host this for friends
  1. API key & CORS: In the Bungie app portal, set Origin Header to * (or your hosted origin, scheme+host only).
  2. Run locally: double-click index.html. Searching works immediately.
  3. Sign-in + sharing: host on GitHub Pages, set OAuth Client Type = Public and Redirect URL to your page URL. See README.md.
