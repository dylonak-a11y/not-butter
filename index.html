First-time setup & how to host this for friends
  1. API key & CORS (required to search): In the Bungie app portal, open your app (ID 47366) and set Origin Header to *. This lets the tool call the API from a browser.
  2. To run locally: just double-click index.html. Searching other players works immediately.
  3. To enable "Sign in with Bungie" + share it: host the file (GitHub Pages is free). Then in the Bungie portal set OAuth Client Type = Public and Redirect URL to your hosted page URL (e.g. https://yourname.github.io/guardian-clock/). See the included README.md for step-by-step hosting.
