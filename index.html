GUARDIAN CLOCK

Destiny 2 time-played tracker

Enter any player's Bungie Name (the name + 4-digit code shown in-game). Their Destiny stats must be public.
Add up to 6 players, then Compare. Fast stats load first; use “Detailed” for streaks (slower).
First-time setup & how to host this for friends
  1. API key & CORS (required to search): In the Bungie app portal, open your app and set Origin Header to * (or your hosted origin, scheme+host only). This lets the tool call the API from a browser.
  2. Run locally: double-click index.html. Searching other players works immediately.
  3. Sign-in + sharing: host the file (GitHub Pages, free). Set OAuth Client Type = Public and Redirect URL to your hosted page URL. See README.md.
